Lesson 1: The Basics of the Web and HTML

How the Web work

When you open your browser and type a certain webpage address like www.google.com, that mean your computer send a request "HTML request" to a server which send back the required HTML document.

The Web and HTML

The Web "WWW": is stand for World Wide Web, also it's a collection of HTML documents.

HTML is stand for Hypertext Markup Language.

HTML documents contain:

  • Text content.
  • Markup (what you look at).
  • Reference to another documents (image, video).
  • Link to anther page.
HTML Attribute

HTML markup contain tags, such as:

  • "b": bold text.
  • "em": Italic text.
  • "a": anchor tag to make a link.
  • "img": to add image.
  • "br": break.
  • "p": paragraph.
Inline and Block Elements

HTML elements either inline or block.

Inline tag such as: a, span, br, img and strong.

Block tags such as: div, p, form.

Lesson 2: Creating a Structure Documents With HTML

HTML, CSS and JavaScript

HTML provides the structure or defines the contents of web pages.

CSS provides the style and the layout of the web pages.

JavaScript provides the interactive elements and program the behavior of web pages.

HTML, CSS and JavaScript are language and they have their own syntax and rules.

The tree-like structure of HTML.

The HTML elements can grow like a family, parents, sons, grandsons and est. Just like a box in another box.

Lesson 3: Adding CSS Style to HTML Structure.

CSS

CSS stand for Cascading Style Sheets.

CSS language allows programmers to control the style of the HTML elements with the same class name.

Their are three ways to add CSS style to a web page:

  • Write CSS code in the head of HTML.
  • Add link to a separate file in the head of HTML.
  • Write the style (inline) with HTML.

To read more about CSS click here.

Avoiding Reputation.

CSS helps programmers to avoid errors by avoiding reputation, especially when they try to edit or change some style in their codes. Instead of changing all their codes they just can change the class style. No need for reputation!
